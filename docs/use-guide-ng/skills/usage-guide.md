# Skill: matechat-ng

`matechat-ng` 是 MateChat Angular 组件库的 Agent Skill，帮助开发者快速构建 AI 聊天界面。通过自然语言描述，Agent skill 将自动获取 MateChat Angular 相关使用文档，完成开发任务。

## 安装方式

下载项目, 将 [matechat-ng](https://gitcode.com/DevCloudFE/MateChat/tree/dev/.trae/skills/matechat-ng) 技能的文件夹复制到你的Agent对应的skills目录下，或执行以下命令安装

```bash
npx skills add DevCloudFE/MateChat
```
